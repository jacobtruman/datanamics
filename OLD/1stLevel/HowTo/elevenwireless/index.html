Taking the Eleven Wireless technology and adding it to the Nomadix technology has changed the face of  what the user will see:

 

 

This is part of the new login screen

 

There are two methods of payment, #1 Bill to Room and #2 Credit Card.

The payment plans are customizable to hourly, daily, weekly or whatever the customer desires.

 

This is a continuation of the login screen:

 

If the customer has created an account, they only need to login with their username and password.

 

If they are wanted internet access they will then either bill it to their room or bill it to their credit card.

 

 

This is the screen to bill to a room. The yellow blank fields on the left are optional, the last name of the person who rented the room and the room number are the only required fields.

 

 

 

 

 

These screenshots are for the credit card purchase option. Only VISA, Mastercard, American Express and Discover are accepted.

 

After Payment method has been chosen and filled out the customer will be required to create a username and password. The username and password currently don’t get deleted when the account runs out of time, the customer can just purchase more time for that user account.

These are the instructions from Eleven Wireless on how to configure these items (These are beta instructions, the final draft is not finished yet):

 

1        Nomadix

Because Nomadix maintains various different menu structures in its different versions and different models, the following configuration guide may not be identical to the Nomadix web interface of your system. Make the following configuration changes to the Nomadix:

1.1         Configuration à AAA

Enable AAA Services

Enable XML Interface

Set XML IP: 69.30.42.70

Enable Origin Server (OS) parameter encoding for Portal Page and EWS

Enable Internal web server

Enable Portal Page

Set Portal Page URL: https://secure.elevenwireless.com/ElevenOS/Portal/

Enable Parameter Passing

Set Portal XML POST URL: http://secure.elevenwireless.com/ElevenOS/Portal/Nomadix_Post.aspx

Set Portal XML Post Port: 80

Submit Changes

1.2         Configuration à RADIUS Client

1.2.1        Server Selection

Select Fixed Routing Mode
Select Default Profile

1.2.2        Miscellaneous Options

Enable URL Redirection.
Enable Send NAS identifier.
Enter a unique location name for NAS Identifier.
Enable Send NAS IP.
Enable Send Framed IP
Enable Byte Count Reset on Account Start

1.2.3        Submit Changes

1.3         Configuration à RADIUS Profiles

1.3.1        Select Default Profile to Edit

1.3.2        Authentication

Enable RADIUS Authentication.
Set primary IP: 69.30.42.74
Set port: 1645
Set Secret Key: ElevenRules

            Set Secondary IP: 69.30.42.75 with identical port and secret key as the primary

1.3.3        Accounting

Enable RADIUS Accounting Service.
Set primary IP: 69.30.42.74
Set port: 1646
Set Secret Key: ElevenRules

Set Secondary IP: 69.30.42.75 with identical port and secret key as the primary

1.3.4        Submit Changes
